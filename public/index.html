FAQ

What are the benefits of using Morph?

Morph is the absolute simplest means to put your idle computer to work mining assets on the PegNet! Setting up your own PegNet mining operation used to be cumbersome and so technical it was reserved only for the race of humans called “system administrators.” No More! Morph is simple to setup and provides you with everything you need to start mining on the PegNet!

Is this just another PegNet mining pool?

Morph uses a different system from mining pools. Morph charges a small administration fee taken every ten blocks, for every other block you will receive full payout upon winning. The difference being that in a mining pool if you were to win a block you would still only get the same as every other block.

Why is there an administration fee for using Morph?

The administration fee covers ongoing miner development, an entry credit subsidy, and the numerous paid API data feeds that are required to run PegNet mining software.

Don’t I need to purchase and set up several technical data feeds for PegNet mining?

Morph removes the hassle and technical complexity out of the PegNet mining days of old! Your installation of Morph includes all the necessary data feeds out of the box! Just install and start mining!

What about Entry Credits (EC)? I read somewhere I need to buy these to enable mining on PegNet?

You won’t believe this, but it’s true! Your installation of Morph includes a complementary supply of Entry Credits to fuel your Morph mining operations!! Happy Mining!!!

Do I need to supply my own PegNet address to accept my mined PEG tokens in Morph?

Morph will automatically produce a PegNet address for you if one is not provided during set up.

Do I need to also setup a Factom Protocol node to support mining on PegNet?

Morph was built to maximize user experience and convenience. Morph’s default settings use Kompendium LLC’s PegNet Open Node (https://pegnetd.com) to support PegNet mining. If you wish to setup and run your own Factom Protocol node to run with Morph, Morph’s settings can easily be adjusted to allow that.

This sounds way too easy and too good to be true – why should I trust Morph Mining?

Morph Mining is an open source project designed and released by Kompendium, LLC a member of the original development team of the PegNet. Additionally, Kompendium LCC is an Authority Node Operator(ANO) and steward of the Factom protocol, on which the PegNet is implemented.
